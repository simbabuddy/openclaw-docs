## 🎯 如何使用

### 作为新用户
1. 从 [入门指南](docs/public/getting-started/) 开始
2. 查看 [用户手册](docs/public/user-guide/) 了解功能
3. 遇到问题？先看 [常见问题](docs/public/faq/)

### 作为团队成员
1. 找到你的 [角色文档](#-角色专属文档)
2. 阅读对应角色的规范和指南
3. 贡献和更新你负责的文档

### 作为贡献者
1. 阅读 [贡献指南](CONTRIBUTING.md)
2. Fork 仓库并创建分支
3. 提交 PR，等待 review
